## 一、产品定位

### 一句话
**选关键词,生成古风艺术照提示词,复制去 AI 绘图工具直接出图。**

### 目标用户
被小红书、公众号古风艺术照刷屏的普通人——看到想拍,但不会写 prompt。

### 核心特点:双模式
- **模板模式**:选一个现成的主题方案,选一个姿势变体,出提示词。适合新手。
- **自由模式**:每个维度自由勾选关键词,多选叠加,模板拼接生成提示词。适合老手和想玩花的人。

## 四、模板模式

### 1. 主题卡片墙
- 手机端:2 列网格,卡片比例 3:4
- 每张卡片:
  - 主题名(宋体字、加粗)
  - 一行氛围描述(淡色小字)
  - 背景:主题色柔和渐变(透明度 15-25%,不刺眼)
  - 右上角小徽章:显示变体数量
- 顶部提供搜索框(仅模板模式需要,42 个主题滚动找太累)

### 2. 变体列表页
- 顶部:返回按钮、主题名、整体氛围描述(30 字内)
- 竖向列表,每个变体:
  - 姿势标签(如"回眸含笑")
  - 一行差异化描述(20 字内)
  - 右侧小箭头

### 3. 结果页(见下方"共用结果页")

## 五、自由模式

### 维度选择页(核心页面)

**7 个维度,每个维度一个折叠区块,默认全部展开**

每个区块展示方式:
- 区块标题(如"场景氛围")
- 关键词以**标签(tag)形式**横向排列,可换行
- 每个标签:未选中是灰底灰字,选中是朱砂红底白字
- 支持**多选**,点一下选中,再点一下取消
- 每个区块右上角显示"已选 X 个"

**7 个维度清单**:

1. **场景氛围**(选地点/环境)
2. **服饰**(选衣服风格)
3. **发型发饰**(选头部造型)
4. **妆容神情**(选脸部表达)
5. **姿势动作**(选身体动作)
6. **镜头构图**(选拍摄角度/景别)
7. **光线质感**(选画面氛围)

### 底部固定栏
- 左侧:显示"已选 X 项"(所有维度总和)
- 右侧:大按钮"生成提示词"(朱砂红、白字)
- 按钮上方微型提示:"建议每个维度至少选 1 项,效果更稳定"
- 如果总选择数 < 3,按钮变灰,文案改为"至少选 3 项"

### 点击"生成提示词"后
- 按照下方"拼接逻辑"拼成完整 prompt
- 跳转到结果页

## 七、视觉设计规范

### 调性
**古风质感 + 现代交互。留白 > 装饰,克制 > 华丽。**

### 色彩

```
主色:墨黑 #1a1a1a(标题、正文)
背景:米白 #faf7f2(全局底色,不用纯白)
点缀:朱砂红 #c1272d(主按钮、选中态)
辅助:淡金 #b8936a(分割线、小印章)
卡片底:#f0ebe3(比背景再深一档)
禁用灰:#d4cec4(未选中标签底色)
```

### 字体
- 中文标题:优先 **霞鹜文楷 LXGW WenKai** 或思源宋体(CDN 加载)
- 中文正文:系统字体(苹方/微软雅黑)
- 数字:无衬线系统字体

### 装饰
- 可以用:极细分割线、小印章、竖向标题装饰线
- **不要用**:祥云、水墨飞溅、花纹边框、中式窗棂

### 动效
- 标签选中:轻微缩放 + 颜色过渡,200ms
- 卡片点击:scale 0.97 + 阴影变化
- 页面切换:左右滑入(手机端)
- 复制成功:按钮文字切换,不用 toast

### 手机端规范
- 按钮最小点击区域 44×44pt
- 正文字号 ≥ 15px,标题 20-28px
- 内容区左右留白 16-20px

## 九、自由模式 · 拼接逻辑

### 拼接模板(骨架句式)

```
古风女子{场景氛围},{光线质感}。
她身着{服饰},{发型发饰},{妆容神情}。
{姿势动作},{镜头构图}。
整体画面{收尾氛围词}。
```

### 拼接规则

1. **多选关键词用"、"连接**
   例:场景氛围选了"红色花林薄雾"和"粉樱雪林" → `古风女子立于红色花林薄雾、粉樱雪林交织的梦境中`

2. **每个维度有前导词**(让语句通顺):
   - 场景氛围:前导"立于"或"置身于"
   - 服饰:前导"身着"
   - 发型发饰:前导"头戴/头梳,簪"
   - 妆容神情:直接接
   - 姿势动作:直接接
   - 镜头构图:前导"画面为"
   - 光线质感:作为收尾氛围,直接接

3. **空维度跳过**
   如果某维度用户没选,直接跳过,不留空白句。

4. **收尾固定兜底**
   最后一句固定加一句氛围收尾:"整体画面细腻、华丽、富有古风写真质感。"

### 示例

用户选择:
- 场景:红色花林薄雾
- 服饰:正红色纱质广袖长裙
- 发饰:赤金流苏发饰
- 姿势:行走中回眸
- 镜头:近景人像
- 光线:电影级光影

拼接输出:
```
古风女子立于红色花林薄雾之中,电影级光影环绕。
她身着正红色纱质广袖长裙,簪赤金流苏发饰。
行走中回眸,画面为近景人像。
整体画面细腻、华丽、富有古风写真质感。
```

## 十一、技术实现建议

- **纯前端单页应用**(React 或 Vue),无后端、无 API 调用
- **数据直接打包在 JSON 文件里**
- **部署**:丢到子域名(如 `prompt.kikiaigc.com`)
- **字体**:霞鹜文楷 CDN,fallback 系统字体
- **复制功能**:`navigator.clipboard.writeText()`,兼容 iOS Safari
- **状态管理**:轻量即可,自由模式的选择态用本地 state

---

## 十二、v1 不做的事

- 用户账号、收藏、历史记录
- 自定义关键词(用户自己加)
- 分享海报生成
- 维度联动(比如选了"战国袍"就自动过滤掉不搭的发饰)
- AI 重写 prompt(保持纯模板拼接,不调 API)
